**Established patterns from Phase 9:**
- Case studies in `src/content/case-studies/{slug}.md`
- Images in `public/images/case-studies/{slug}/` with naming: `hero.png`, `image-01.png`, `image-02.png`, etc.
- Frontmatter schema: title, description, date, featured, confidentiality, tags, coverImage
- Absolute image paths from /public: `/images/case-studies/{slug}/hero.png`
- Inline images referenced with markdown: `![Alt text](/images/case-studies/{slug}/image-NN.png)`

**Schema requirements (from config.ts):**
- title: string (required)
- description: string (required)
- date: date (required)
- featured: boolean (default: false)
- confidentiality: enum 'public' | 'partial' | 'nda' (default: 'public')
- tags: array of strings (default: [])
- coverImage: string (optional, absolute path from /public)
