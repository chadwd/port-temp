## Overview

Initial MVP of a content-first portfolio website for a senior/principal product designer. The site features a calm, minimal, editorial aesthetic with adaptive light/dark mode, markdown-driven content management, and case study confidentiality levels. Built with Astro and deployed to GitHub Pages with custom domain.

## Milestone Summary

**Key Decisions:**

- Astro framework - Fast, markdown-first, minimal JS
- GitHub Pages hosting - Free, controlled, no vendor lock-in
- Markdown + frontmatter - Easy to write, version controlled, portable
- Adaptive color mode - Respects user preference
- Slate blue accent (#334155) - Professional, not corporate
- System fonts only - Performance priority
- Warm color palette - Grounded feel, not cold
- Subtle confidentiality badges - Professional look
- withastro/action@v3 only - Handles build + upload together

**Issues Resolved:**

- Fixed duplicate artifact upload in GitHub Actions workflow (409 conflicts)
- Resolved GitHub Pages enablement via API
- Configured custom domain and DNS for apex domain deployment
